# 🔀 Atividade Estrutura de Controle DIO

Aplicação Java de console desenvolvida como atividade do bootcamp da DIO. O projeto reúne 4 exercícios práticos com foco no uso de estruturas de controle, como `if/else`, `switch` e `do/while`.

---

## 🗂️ Exercícios

**[1] Tabuada**
O usuário escolhe um número e o programa exibe a tabuada completa dele, do 1 ao 10.

**[2] Calculadora de IMC**
O usuário informa seu peso e altura. O programa calcula o IMC com base na fórmula padrão e retorna uma classificação (como abaixo do peso, normal, sobrepeso, obeso, etc.) utilizando estruturas `if/else`.

**[3] Contagem Decrescente**
O usuário escolhe dois números. O programa parte do maior e faz uma contagem decrescente até o menor, com a opção de exibir apenas os números **pares** ou **ímpares** ao longo do caminho.

**[4] Verificador de Divisibilidade**
O usuário informa dois números para verificar se um é divisível pelo outro. Caso o divisor seja 0, o programa continua solicitando um novo número até que um valor válido seja informado — evitando a divisão por zero.

---

## 🛠️ Tecnologias Utilizadas

- **Java 25 no IntelliJ**

---

## 📚 Contexto

Este projeto foi desenvolvido como parte de uma atividade prática do bootcamp da [DIO (Digital Innovation One)](https://www.dio.me/), com o objetivo de praticar estruturas de controle em Java, incluindo:

- Condicionais com `if`, `else if` e `else`
- Seleção múltipla com `switch`
- Repetição com `do/while`
- Validação de entrada do usuário
